# ProyectoDispenser
Este repositorio contiene el firmware y la documentación de ingeniería para un Dispensador Automático de Comida para Mascotas desarrollado utilizando la plataforma Arduino.

El proyecto se enfoca en la aplicación práctica de conceptos de control automático para garantizar una dosificación de alimento precisa y programada, mejorando la calidad de vida de las mascotas.
